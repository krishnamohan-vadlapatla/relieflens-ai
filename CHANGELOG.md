# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-08

### Added

- Initial release for Gemma 4 Good Hackathon
- Flutter mobile app with dark theme UI
- FastAPI backend with Ollama/Gemma 4 integration
- 5 core features:
  - AI Disaster Image Analyzer
  - Multilingual Voice Emergency Assistant
  - RAG Knowledge System
  - Coordination Dashboard
  - Confidence + Safety Layer
- Multilingual support (Telugu, Hindi, English)
- Offline-first architecture
- Comprehensive documentation
- Demo scripts and sample data

### Features

- Image upload and analysis
- Voice recording with waveform visualization
- Semantic search for emergency knowledge
- Real-time coordination dashboard
- Confidence scores on all AI responses
- Safety verification prompts

### Technical

- Gemma 4 2B/7B model support
- Ollama local inference
- Whisper.cpp speech-to-text
- ChromaDB vector storage
- Riverpod state management
- Flutter 3.x cross-platform
